Notificar Vendedor - UC013

O Ator deve notificar o Vendedor quando uma venda for realizada.

Ator:

  • Sistema

Pré-condições:

  • O Ator precisa que uma compra seja finalizada.
  • O Ator precisa saber a forma de pagamento.
  • O Ator precisa saber a forma de retirada

Pós-condições:

  • O Vendedor deve estar notificado da venda realizada.
