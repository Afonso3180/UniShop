Calcular Total - UC011

O sistema deve calcular o valor da compra, contabilizando todos os produtos que o comprador deseja obter e que estão localizados no carrinho de compra.

Ator:

  • Usuário

Pré-condições:

  • O comprador deve estar logado no sistema
  • O produto deve estar no carrinho de compra

Pós-condições:

  • O valor total deve aparecer ao comprador
