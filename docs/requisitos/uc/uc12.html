Finalizar Compra - UC012

O usuário deve finalizar a compra após escolher os produtos, escolher o local de entrega e efetuar o pagamento caso seja no cartão.

Ator:

  • Usuário

Pré-condições:

  • O comprador deve estar logado no sistema
  • O comprador deve estar realizando uma compra
  • O comprador deve atender a todos os requisitos para finalizar a compra

Pós-condições:

  • O vendedor deve entregar o(s) produto(s)
  • O comprador poderá efetuar outra compra
