Recuperar Senha - UC003

Caso o usuário tenha esquecido a senha, ele será capaz de recuperar o acesso ao sistem criando outra senha.

Ator:

  • Usuário

Pré-condições:

  • Usuário precisa estar na página de login.

Pós-condições:

  • O usuário terá acesso ao sistem novamente;
  • A senha para acesso será diferente necessariamente da anterior.
