Notificar Comprador - UC014

O Sistema deverá notificar o comprador quando o vendedor chegar ao local de entrega do produto, que foi decidido entre as duas partes.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema
  • O vendedor deve ter aceitado a compra

Pós-condições:

  • O comprador irá receber o produto do vendedor no local escolhido
