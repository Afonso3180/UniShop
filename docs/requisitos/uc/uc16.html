Confirmar Pagamento - UC016

O Ator precisa informar se o pagamento foi efetuado.

Ator:

  • Sistema

Pré-condições:

  • O Ator precisa que uma compra seja finalizada.
  • O Ator precisa saber a forma de pagamento.
  • O Ator precisa saber a forma de retirada

Pós-condições:

    O status do pedido deve ser: pago.
