Confirmar Pagamento - UC016

O usuário pode confirmar o pagamento após inserir os dados do cartão e processar o pagamento.

Ator:

  • Sistema

Pré-condições:

  • O usuário precisa estar logado no sistema.
  • O carrinho de compras do comprador precisa estar definido.

Pós-condições:

  • O vendedor deve ser notificado.
