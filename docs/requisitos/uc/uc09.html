Escolher Forma de Pagamento - UC009

O usuário deve escolher qual forma de pagamento para a compra que está sendo realizada, o comprador pode pagar no cartão de crédito ou débito ou no dinheiro.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema
  • O usuário deve ter adicionado algum produto ao carrinho

Pós-condições:

  • O usuário terá que escolher o local de entrega
  • O usuário pode desfazer a forma de pagamento
