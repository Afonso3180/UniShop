Listar vendedores - UC005

O sistema mostrará uma lista para o comprador que poderá escolher o vendedor que deseja para comprar o produto, e aparecerá os produtos desse vendedor.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema
  • Deve haver algum vendedor cadastrado no banco de dados

Pós-condições:

  • O usuário poderá escolher os produtos do vendedor
  • O usuário poderá visualizar as informações do vendedor
