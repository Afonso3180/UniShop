Listar Produtos - UI/UX - 004

O usuário pode listar diversos produtos cadastrados pelos vendedores, onde é possível ver o valor, sabor e descrição.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema
  • Deve haver algum produto cadastrado por um vendedor

Pós-condições:

  • O usuário poderá escolher o produto
  • O usuário pode consultar a descrição do produto
