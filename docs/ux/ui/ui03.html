Cadastrar Produto - UI/UX - 003

O vendedor após ter sido cadastrado no sistema, tem a opção de cadastrar os seus produtos para venda, onde será cadastrado o nome do produto, valor.

Ator:

  • Usuário

Pré-condições:

  • Usuário precisa estar logado

Pós-condições:

  • O produto poderá ser visto pelos compradores
  • O vendedor poderá adicionar outros produtos
