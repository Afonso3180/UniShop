Documentação UniShop

1. Objetivo

Desenvolver um Sistema de marketplace, denominado UniShop. Com a finalidade de facilitar e simplificar a interação e o processo de venda de produtos entre os universitários da EST (UEA). Este documento visa apresentar características e detalhes do sistema.
